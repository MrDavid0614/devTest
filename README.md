# Dev Test

Technologies used:

- ReactJS
- NodeJS
- WebSockets
- Rest API
- SQLite

In order to run this project, you need to follow some few steps after clone the repo and move inside the directory:

## Run the backend:

Being in the root directory, run this commands:

```bash
    cd backend
```

```bash
    yarn
```

OR

```bash
    yarn install
```

```bash
    node index.js
```

## Run the frontend:

Being in the root directory, run this commands:

```bash
    cd frontend
```

```bash
    yarn
```

OR

```bash
    yarn install
```

```bash
    yarn dev
```
